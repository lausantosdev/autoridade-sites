# Commit Push — Finalização do Ciclo

Você está finalizando um ciclo de desenvolvimento. Faça commit e push das mudanças implementadas.

## Pré-requisito

Só execute essa skill se o Sonnet retornou **APROVADO** ou **APROVADO COM RESSALVAS** no audit-execution. Se o status for REQUER CORREÇÃO, não faça commit — volte para execute-spec.

## Input esperado

```
LOG DE EXECUÇÃO:
[output do execute-spec]

RESULTADO DA AUDITORIA:
[output do audit-execution]
```

## Processo

1. Rode `git status` para confirmar os arquivos alterados
2. Rode `git diff --stat` para revisar o escopo do que será commitado
3. Verifique se há arquivos que **não** deveriam estar no commit (fora do escopo da Spec)
4. Stage apenas os arquivos listados na Spec
5. Gere a mensagem de commit
6. Faça o commit
7. Faça o push

## Formato da mensagem de commit

Use o padrão Conventional Commits derivado do conteúdo real da Spec:

```
<tipo>(<escopo>): <descrição curta em inglês, imperativo, max 72 chars>

<corpo opcional: o que foi feito e por quê, max 3 bullets>

Co-Authored-By: Gemini <noreply@google.com>
```

**Tipos:** `fix` (bug), `feat` (nova funcionalidade), `refactor` (sem mudança comportamental), `chore` (infra/config), `test` (testes)

**Regras da mensagem:**
- Derive do objetivo da Spec — não invente
- Imperativo: "add", "fix", "remove" — não "added", "fixing"
- Sem "misc", "updates", "changes" — seja específico
- Se a auditoria registrou ressalvas, mencione no corpo

## Checklist antes de executar

- [ ] `git status` confirma apenas arquivos do escopo da Spec
- [ ] Nenhum arquivo `.env`, de config sensível, ou fora do escopo está staged
- [ ] Auditoria do Sonnet está APROVADO ou APROVADO COM RESSALVAS
- [ ] Branch está atualizada com a remota (`git pull` se necessário)

## Comandos a executar em sequência

```bash
git status
git diff --stat
git add [arquivos específicos da Spec — nunca git add .]
git commit -m "$(cat <<'EOF'
<mensagem gerada>
EOF
)"
git push
```

**Nunca use `git add .` ou `git add -A` — adicione arquivos explicitamente pelo nome.**
